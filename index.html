// 核心功能

为工程计算而生

⚡
开箱即用，零配置

独立 .exe 可执行程序，无需安装 Python 或任何依赖环境。下载解压，双击立即运行，最低化启动成本。

📊
CSV 批量预测

支持数百组特征参数一键导入，并发推演全部样本，结果即时导出。内置可视化引擎，自动绘制标况对比特征曲线。

🛡️
物理防呆机制

底层算法融合严谨燃烧学物理约束，自动规避违反热力学的异常输出，对每条结果提供高 / 中 / 低三级置信度评级。

🔬
多算法集成

内置 LightGBM、XGBoost、随机森林、SVR 等 12+ 候选算法，自动筛选最优集成策略，兼顾精度与泛化能力。

📈
模型诊断面板

内置诊断模块可查看 R²、RMSE、MAE 等多维评估指标，以及特征重要性分布图，支持深度结果溯源。

🌡️
宽域工况覆盖

支持大范围当量比、初始压力与温度参数组合推演，适用于发动机设计、安全评估与爆炸极限分析等多种场景。

运行须知 · Windows 安全提示

本软件为科研团队独立开发打包，未添加商业数字签名。首次运行时可能触发 Windows SmartScreen 或第三方杀毒软件的误报提示，属于正常现象，程序本身无任何恶意行为。

1
遇到蓝色 SmartScreen 弹窗时，点击 "更多信息"
2
点击 "仍要运行" 即可正常启动
3
若为杀毒软件提示，选择 "允许" 或加入信任白名单
